# FinalProject
Tetris Game

To compile and run: 
  javac *.java
  java Main
  
How to use:
  After compiling and running the game, the player will be taken to a start screen. There the player can press any key to begin the game. Our tetris game interacts with the keyboard to move the falling block. To rotate the block 90 degrees clockwise, press the up arrow. To move the block to the right or left, press the right or left arrow.
